## Como rodar

### 1 - Faça o `git clone` do repositório do github

```shell
git clone https://github.com/gabrielfelipedy/monitoramento_switches
```


### 2 - Instale as dependências

```nginx
npm install
```

### 3 - Execute na máquina local

```nginx
ng serve
```

caso queira servir para toda a rede local:

```nginx
ng serve --host 0.0.0.0
```
